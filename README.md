# Anime Kingdom Login Page

An anime-themed login page with stunning visual effects, animations, and video playback.

![Anime Kingdom Login](imagepro.jpg)

## Features

- **Anime-Inspired Design**: Beautiful anime-themed interface with Japanese typography
- **Visual Effects**: Blur effects, hover animations, and smooth transitions
- **Interactive Elements**: Animated login form with floating labels
- **Video Playback**: Horizontal full-screen video playback with sound after login
- **Responsive Design**: Looks great on various screen sizes

## Technologies Used

- HTML5
- CSS3 (with animations and transitions)
- JavaScript (vanilla)
- Google Fonts (Noto Sans JP for Japanese text)
- Font Awesome (for social media icons)

## File Structure

- `index.html` - Main HTML structure
- `styles.css` - All styling and animations
- `script.js` - JavaScript functionality
- `imagepro.jpg` - Background image
- `videopro.mp4` - Video that plays after login

## How It Works

1. Enter username and password in the login form
2. Click the "Login" button
3. After successful login, an "Enter the Kingdom" button appears
4. Click the button to start the full-screen horizontal video with sound

## Installation

Simply clone or download this repository and open `index.html` in your web browser.

```bash
git clone https://github.com/Rudraa01/Anime_login_page_idea.git
cd Anime_login_page_idea
```
